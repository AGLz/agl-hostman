# Frontend Components

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend components.

## When to use this skill:

- Creating new UI components (React, Vue, Svelte, Web Components)
- Defining component props, interfaces, or prop types
- Implementing component composition and nesting
- Managing component state (useState, data, local state)
- Building reusable component libraries
- Extracting common UI patterns into shared components
- Documenting component APIs and usage examples
- Implementing component variants or configuration options
- Setting up default props or sensible defaults
- Working with component files like `Button.tsx`, `Card.vue`, `components/`, `*.component.js`
