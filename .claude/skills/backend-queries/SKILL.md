# Backend Queries

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle backend queries.

## When to use this skill:

- Writing database queries using SQL or ORM query builders
- Implementing data fetching and filtering logic
- Optimizing queries for performance
- Setting up eager loading or joins to prevent N+1 queries
- Using SELECT statements with specific columns (avoiding SELECT *)
- Implementing query transactions for data consistency
- Adding query timeouts and error handling
- Setting up query result caching
- Working with query builders, repositories, or service layers
- Writing parameterized queries to prevent SQL injection
- Working with files in `queries/`, `repositories/`, `services/`, or query-related modules
