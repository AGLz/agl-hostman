# Frontend Responsive

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend responsive.

## When to use this skill:

- Implementing mobile-first responsive designs
- Creating layouts that adapt to different screen sizes
- Setting up breakpoints for mobile, tablet, and desktop views
- Using responsive utility classes (Tailwind, Bootstrap)
- Writing media queries for custom responsive behavior
- Optimizing touch targets for mobile devices (minimum 44x44px)
- Implementing responsive navigation menus
- Ensuring readable typography across all screen sizes
- Optimizing images and assets for mobile networks
- Testing layouts across multiple device sizes
- Using relative units (rem, em) instead of fixed pixels
