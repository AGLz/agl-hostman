# Frontend CSS

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend CSS.

## When to use this skill:

- Writing or modifying CSS styles for components
- Working with CSS frameworks (Tailwind CSS, Bootstrap, Material UI)
- Defining design tokens (colors, spacing, typography)
- Setting up utility classes or style systems
- Implementing responsive styles and breakpoints
- Optimizing CSS for production (purging, tree-shaking)
- Avoiding unnecessary custom CSS or framework overrides
- Creating or updating CSS modules or styled-components
- Configuring CSS preprocessors (Sass, PostCSS)
- Working with files like `styles.css`, `tailwind.config.js`, `*.module.css`, `styled.js`
