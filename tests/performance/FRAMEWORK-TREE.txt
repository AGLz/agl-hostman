Test Script Features:
======================

All test scripts include:
  ✓ Color-coded output (BLUE info, GREEN success, YELLOW warning, RED error)
  ✓ JSON result format for machine parsing
  ✓ Statistical analysis (avg, min, max, p50, p95, p99)
  ✓ Status determination (GOOD/WARNING/CRITICAL based on thresholds)
  ✓ Flexible configuration via environment variables
  ✓ Comprehensive error handling and validation
  ✓ Dependency checking with helpful error messages
  ✓ Verbose mode for detailed output
  ✓ Help documentation

Common Environment Variables:
  RESULTS_DIR    - Custom results directory (default: /tmp/performance-results)
  DURATION       - Test duration in seconds (varies by test)
  VERBOSE        - Enable verbose output (0 or 1)

Quick Usage Examples:
=====================

# Run individual tests
./baseline/system-baseline.sh
./network/wireguard-perf.sh
./storage/nfs-benchmark.sh
./services/archon-perf.sh

# Run with custom settings
DURATION=60 ./network/wireguard-perf.sh
TEST_SIZE=5G ./storage/nfs-benchmark.sh
REQUEST_COUNT=500 CONCURRENCY=50 ./services/archon-perf.sh

# Run complete suite
./run-performance-suite.sh
./run-performance-suite.sh --category network
./run-performance-suite.sh --category "baseline storage"

# View results
cat /tmp/performance-results/*.json | jq '.'
cat docs/test-reports/performance/*.md
