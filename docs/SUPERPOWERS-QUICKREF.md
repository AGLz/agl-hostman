## 🚀 Quick Start

### Invoke a Skill
```
Using [skill-name] to [purpose]
```

Then use the Skill tool with the skill name.

---

## 📋 Essential Skills (Use Daily)

### Before ANY Work
- **using-superpowers** - Discover applicable skills
- **brainstorming** - Explore approaches before coding
- **test-driven-development** - Write tests first

### During Development
- **systematic-debugging** - Fix bugs methodically
- **executing-plans** - Follow implementation plans
- **subagent-driven-development** - Spawn specialized agents

### Before Completing
- **verification-before-completion** - Confirm work is done
- **requesting-code-review** - Get code reviewed
- **finishing-a-development-branch** - Prepare for merge

---

## ⚠️ Golden Rule

**If there's even 1% chance a skill applies, YOU MUST USE IT.**

No exceptions. No rationalizing.

---

## 🔧 Common Workflows

### Starting a New Feature
```
1. using-superpowers (discover skills)
2. brainstorming (explore approaches)
3. writing-plans (create plan)
4. test-driven-development (TDD)
5. verification-before-completion (verify)
6. requesting-code-review (review)
```

### Fixing a Bug
```
1. systematic-debugging (diagnose)
2. test-driven-development (write test)
3. verification-before-completion (verify)
```

### Parallel Tasks
```
1. dispatching-parallel-agents (coordinate)
2. subagent-driven-development (execute)
```
