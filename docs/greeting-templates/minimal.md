# Minimal Greeting Templates

## Template Structure (60 tokens avg)

### Standard Format
```
[ACTION VERB] + [SCOPE]
```

### Examples

#### Simple Acknowledgment
```
"Understood. I'll proceed with the implementation."
```

#### Status Update
```
"Continuing with security analysis. Found 3 issues so far."
```

#### Session Continuation
```
"I'll fix the identified vulnerabilities now."
```

#### Quick Response
```
"Deploying the hotfix to production."
```

## Token Budget

| Component | Tokens |
|-----------|--------|
| Action statement | 45-80 |
| **Total** | **45-80** |

## When to Use

✅ **Use Minimal Template**:
- Active session continuation
- Well-defined routine tasks
- Quick acknowledgments
- Status updates
- Simple operations

❌ **Use Balanced Instead**:
- New task initiation
- Complex workflows
- Multi-agent coordination
- Unknown scope

⚠️ **Avoid Minimal When**:
- User has asked questions (indicates confusion)
- Task complexity is unclear
- Coordination is required
- Context has been lost
