## Usage Instructions

### How to Use This Template

1. **Copy**: Copy this entire template
2. **Rename**: Save as `AGL-XX-COMPLETION-REPORT.md`
3. **Fill**: Complete all applicable sections
4. **Review**: Have team review and provide feedback
5. **Store**: Save in `/docs/linear/` directory
6. **Reference**: Link from task tracking system

### Section Completion Guidelines

**Required Sections** (Must complete):
- Task Information
- Executive Summary
- Implementation Details (all phases)
- Testing & Validation
- Documentation
- Sign-off

**Optional Sections** (Complete if applicable):
- Technical Changes
- Metrics & Performance
- Deployment
- Issues & Blockers
- Lessons Learned
- Next Steps

### Tips for Effective Reports

1. **Be Specific**: Use concrete numbers and metrics
2. **Provide Evidence**: Link to logs, screenshots, test results
3. **Document Decisions**: Explain why choices were made
4. **Include Metrics**: Before/after comparisons
5. **Capture Learnings**: What to replicate/improve
6. **Next Steps**: Clear handoff to follow-up work
