# Diagramas de arquitectura

Saída de `agl-architecture-diagram` e `drawio-skill`.

| Formato         | Uso                             | Ferramenta                      |
| --------------- | ------------------------------- | ------------------------------- |
| `.md` + Mermaid | ADRs, versionado no Git         | Cursor / qualquer editor        |
| `.drawio`       | Edição visual, fluxos complexos | draw.io desktop ou diagrams.net |

Convenção de nomes: `<feature>-flow.md`, `<serviço>-architecture.drawio`.
