# Global Tech Stack

This Skill provides Codex with specific guidance on how to adhere to coding standards as they relate to how it should handle global tech stack.

## When to use this skill:

- Referencing approved technologies and frameworks for the project
- Making decisions about adding new dependencies or libraries
- Setting up project infrastructure and deployment pipelines
- Configuring development tools and build systems
- Ensuring consistency with project's chosen technology stack
- Documenting technology choices and rationale
- Onboarding new team members to the tech stack
- Evaluating alternatives or upgrades to existing technologies
- Working with package managers, dependency files, or infrastructure config
- Creating technical documentation that references the stack
