# Backend API

This Skill provides Codex with specific guidance on how to adhere to coding standards as they relate to how it should handle backend API.

## When to use this skill:

- Creating or modifying REST API endpoints and routes

- Implementing API versioning strategies
- Designing URL structures and resource naming
- Adding query parameter support for filtering, sorting, or pagination
- Setting up rate limiting and API throttling
- Configuring HTTP status codes and error responses
- Writing API documentation or OpenAPI/Swagger specs
- Implementing nested resource endpoints
- Working with files like `routes.js`, `api.py`, `controllers/`, `endpoints.ts`, `app.js`
