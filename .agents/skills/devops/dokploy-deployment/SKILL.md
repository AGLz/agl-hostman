# Dokploy Deployment

This Skill provides Codex with specific guidance on deploying Laravel applications with Dokploy.

## When to use this skill:

- Deploying Laravel applications to Dokploy platform
- Configuring Dokploy projects and applications
- Setting up Docker Compose-based deployments
- Managing environment variables and secrets in Dokploy
- Configuring custom domains and SSL certificates
- Setting up MySQL, PostgreSQL, and Redis services
- Configuring persistent storage for application data
- Implementing health checks and uptime monitoring
- Setting up git-based auto-deployments
- Configuring deployment webhooks and notifications
- Implementing blue-green deployment strategies
- Scaling applications with multiple replicas
- Viewing and aggregating application logs
- Managing application versions and rollbacks
- Configuring multi-environment deployments
- Integrating Dokploy with GitHub/GitLab CI/CD
- Setting up custom build and deployment scripts
- Troubleshooting deployment failures and issues
